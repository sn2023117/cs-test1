## 📌 Problem Statement

대학 생활 중 공강 시간이 생겼을 때 "집에 다녀올 수 있을까?"를 계산하는 게 귀찮고 불편했다.  
버스 시간, 왕복 이동 시간, 집에 있을 수 있는 시간, 교통비를 각각 따로 계산해야 해서  
정작 결정을 못 하고 시간을 낭비하는 경우가 많았다.

## 💡 Solution

공강 시간과 이동 수단을 입력하면 왕복 시간 + 여유 시간 + 교통비를 한 번에 계산해주고  
"다녀올 수 있는지 없는지"를 바로 알려준다.  
추가로 주변 편의시설 추천과 위치별 할 일 관리까지 제공한다.

---

## 🛠️ 핵심 기능

| 기능 | 설명 |
|------|------|
| 🚌 이동 시간 계산 | 이동수단별 왕복시간·여유시간·행동 추천 |
| 💰 교통비 계산 | 환승할인 포함 왕복 요금 자동 계산 |
| 📍 주변 편의시설 | 공강시간 기준 방문 가능/불가 필터링 |
| 📋 위치별 할 일 | 장소·우선순위·식사여부 연동, 로컬 저장 |
